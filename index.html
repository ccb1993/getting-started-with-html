Important Concepts

World Wide Web: How it works

-Many computers communicate with eachother
-When someone goes to a web page, an HTTP Request is sent to a server (computer at another location hosting web files)
-Servers send back appropriate HTML documents to fulfill requests sent by user computers
-Computers interpret HTML documents via browsers

HTML
"Hyper Text Markup Language"

-HTML documents are the basis of the web and form most web content
-text content is "what you see"
-markup is "what it looks like"
-used to reference other documents and provide links to other pages
-regular text in HTML will appear as just regular text

Tags and Elements

-HTML documents consist of HTML elements
-Elements consist of tags (opening at the beginning, closing at the end) which define the properties of text inside HTML documents

Inline vs. Block Elements

Inline elements differ from block elements as they just stay in line whereas block elements form a "box" around what is inside them making the content inside a package deal with regards to where it is located on a page.
